# eSports-Event-Management

SPP Laboratory Works

##Topic
eSports Event Organisation And Management System

##Requirements
NPM, Bower, Gulp installed.

##Deploy
1. Pull Contents Of This Repo
2. CD to the project dir and type:
```
cd ./web-compiler
npm install
bower install
```
